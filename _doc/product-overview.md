# Product Overview

## What It Is
A personal AI image generation workspace that lets the owner create images across the full creative spectrum using their own API key. This is a private, single-user tool — not a public platform — giving direct access to the generation capabilities of the connected API provider without additional app-level restrictions.

## Users
- **Primary user**: The owner/creator — one individual using this as a personal creative workspace.

## Positioning
A clean, fast, private workspace for personal AI image generation. The user brings the API key (Stability AI, Replicate, or a compatible provider); the app handles prompting, generation, gallery, and history. No app-level content filters are added beyond what the connected API enforces. No subscription tiers, no artificial usage caps beyond what the API itself imposes.

## Core Capability (MVP)
- Prompt input → image generation via user-supplied API key
- Personal gallery to browse, save, and organize generated images
- API key configuration (securely stored, easily updated, switchable between providers)
- Generation history with prompt details for every past creation

## Brand & Tone
- Personal, creative, uncluttered
- Calm and focused — a maker's workspace, not a consumer product
- Name: To be confirmed during onboarding

## Scope & Constraints
- Single user (personal workspace, not multi-tenant)
- User supplies and manages their own API key and bears its associated generation costs
- Content governance is the responsibility of the connected API provider, not the app
- No in-app usage limits or caps beyond what the API provider enforces
